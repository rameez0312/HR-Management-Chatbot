

# 🤖 HR Management Chatbot

An intelligent **HR Management Chatbot** built with Python that combines **HRMS functionalities** with conversational capabilities.  
This system helps automate employee interactions, manage HR operations, and provide quick responses to HR-related queries.

---

## 🚀 Features

### 🧑‍💼 HR Management System
- Employee management  
- Leave request handling  
- Meeting scheduling  
- Ticket / issue tracking  

### 🤖 Chatbot Capabilities
- Responds to HR-related queries  
- Automates repetitive HR tasks  
- Provides quick and consistent answers  

### ⚙️ Backend Design
- Modular Python architecture  
- Scalable and easy to extend  
- Environment-based configuration
